# MUON
An implementation of the MUON optimizer, applied for a classifier of the CIFAR-10 dataset, with tests to compare with AdamW.

## Setup

### Required packages
```
uvicorn
python
```

Run from root
```bash
bash scripts/setup.sh
```

## Run

### Train models
```bash
bash scripts/train.sh
```

### Test models
```bash
bash scripts/test.sh
```
